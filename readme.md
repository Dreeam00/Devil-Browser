# DevilBrowser

## 概要

このプロジェクトは、**「世界で一番使いたくないブラウザ」** を目指して作成されました。ユーザーの操作を妨害し、予測不能な動作をすることで、ブラウザ利用における新たな（そして不快な）体験を提供します。

## 主な嫌がらせ機能

DevilBrowserは、以下のような様々な方法でユーザーを不快にさせます。

### 1. 起動時

*   **偽のアップデート**: 起動時に「アップデート中...」というダイアログが表示され、5秒後にランダムな結果になります。
    *   10%の確率で「アップデート失敗。ブラウザを終了します。」と表示され強制終了。
    *   90%の確率で「アップデート成功！（何も変わりませんが）」と表示され起動。

### 2. ウィンドウ操作

*   **閉じるボタンの反逆**: ウィンドウの閉じるボタン（Xマーク）をクリックすると、ブラウザが終了する代わりに新しいタブが10個開きます。

### 3. アドレスバーの欺瞞

*   **URLの嘘**: URLバーに入力してEnterを押すと、以下のような予測不能な動作をします。
    *   70%の確率で、入力したURLとは全く異なるサイト（Google, Bing, Yahoo, Wikipedia, ニコニコ動画, YouTubeのいずれか）に強制的に移動します。
    *   15%の確率で、ローカルファイル `C:/Windows/win.ini` を開きます。
    *   10%の確率で、設定画面に強制的に遷移します。
    *   5%の確率で、何も起きずに「URLバーは働かない時もあります。人生とはそういうもの。」というメッセージが表示されます。
*   **過剰な警告**: サイトアクセス前に、2〜5回の「このサイトは危険です（理由：なんとなく）」といった内容の警告ダイアログが連続して表示されます。

### 4. 検索機能の混乱 (OopsSearch Engine)

*   **クエリの改ざん**: 検索クエリ中の特定の単語（例:「猫」）が勝手に別の単語（例:「犬」）に変換されます。
*   **検索エンジンのたらい回し**: 「Google」「Bing」「Yahoo」といった検索エンジン名を含むクエリは、次の検索エンジンに強制的にリダイレクトされ、最終的には「Ask Jeeves」での検索が404エラーに繋がります。
*   **予測不能な結果**:
    *   10%の確率で、検索実行時にブラウザが強制終了します。
    *   20%の確率で、「404 - Not Even a Fake Page Found」というエラーページが表示されます。
    *   残りの確率で、検索クエリに対して「なぜ検索するのですか？自分で考えられないのですか？」といった「質問返し」をするか、全く無関係でクリックできない「ランダムな結果」を表示します。

### 5. ナビゲーションとタブ

*   **「進む」ボタンの自殺**: 「進む」ボタンをクリックすると、ブラウザが強制終了します。
*   **「更新」ボタンの無気力**: 「更新」ボタンをクリックすると、現在のタブのコンテンツが「画面が真っ白になりました。これが更新です。」というメッセージに置き換わります。
*   **「広告ブロック」ボタンの悪意**: 「広告ブロック」ボタンをクリックすると、現在のページの広告らしき要素（`DIV`, `P`, `IMG`タグ）をランダムに選んで5倍に増殖させます。
*   **タブの自動複製**: 1分ごとに、現在アクティブなタブが複製されて新しいタブとして追加されます。
*   **履歴の破壊**: 履歴の追加時に25%の確率で、偽のURLが挿入されたり、履歴が削除されたり、履歴の順序がシャッフルされたりします。
*   **履歴アクセスのフリーズ**: 「戻る」または「進む」ボタンをクリックした際に、15%の確率で3〜8秒間ブラウザがフリーズします。

### 6. 設定画面の誘惑と罠

*   **「地獄の設定」**: 設定ボタンをクリックすると、「地獄の設定」と題されたダイアログが開きます。
*   **閉鎖が死を招く**: この設定ダイアログを閉じると、ブラウザ全体が強制終了します。
*   **押せない設定**: 設定項目はツリー形式で表示されますが、ほとんどの項目は選択できません。
*   **押せる罠**: まれに「押せそう…？」と表示される項目があり、それをクリックするとランダムな「致命的な設定エラー」ダイアログが表示された後、ブラウザが強制終了します。

### 7. UIの嫌がらせ

*   **スクロールの反転**: 各タブのコンテンツの垂直・水平スクロールが逆方向に動作します。
*   **ボタンの回避行動**: ナビゲーションボタン、検索ボタン、設定ボタンなどにマウスカーソルが乗ると、ボタンが操作しづらくなるように検索フィールドにフォーカスが移ります。
*   **強制的なタイプミス**: アドレスバーや検索フィールドへのキー入力時、15%の確率で入力文字が別のランダムな英数字に勝手に置き換わります。
*   **バックスペースの裏切り**: バックスペースキーを押しても、50%の確率でランダムな文字が挿入され、入力内容を修正できません。
*   **永遠のローディング**: ブラウザウィンドウにマウスカーソルが乗ると、常にローディングカーソル（砂時計/スピナー）に変化します。
*   **予測不能なテーマ**: 20秒ごとに、アクティブなタブの背景色と前景色、およびアドレスバー、検索フィールド、ボタン、プログレスバーの色がランダムに変化します。
*   **不快なビープ音**: 20秒〜90秒のランダムな間隔で、システムのビープ音が鳴り響きます。
*   **言語の混乱**: 15秒ごとに、主要なUI要素（ボタンテキスト、アドレスバー・検索バーのプレースホルダーテキスト）の表示言語や内容がランダムに変化し、ユーザーを混乱させます。
*   **単語の消失**: 10秒ごとに、アクティブなタブのコンテンツからランダムな単語が自動的に削除されます。

### 8. ページ読み込み中のトラップ

*   **突然のクラッシュ**: ページ読み込み中に5%の確率でブラウザが強制終了します。
*   **広告の増殖**: ページ読み込み完了後、現在のページの `DIV`, `P` 要素の中からランダムに1〜2個を選び、それぞれを2倍に増殖させます。
*   **偽の完了**: 50%の確率で、代わりに「お得な情報！今すぐここをクリック！」という偽の広告ページを表示し、ビープ音を鳴らします。残りの50%では「読み込み完了!...のはずだった。」というメッセージを表示します。
*   **進まないプログレスバー**: プログレスバーが99%で止まることがあります。

## 使用技術

*   Java Swing

## ビルドと実行方法

1.  ソースコードをコンパイルします。
    ```bash
    javac -d . DevilBrowser.java OopsSearchEngine.java SettingsDialog.java
    ```
2.  アプリケーションを実行します。
    ```bash
    cd ..
    java -cp Devil-Browser-main DevilBrowser
    ```

さあ、始めましょう

DevilBrowserとの新たな関係を始めるための手順はこちらです。ご利用にはJava開発環境 (JDK) と、少しの覚悟が必要です。

ビルドプロセス

srcディレクトリにあるソースコードから、以下のコマンドでアプリケーションをビルドします。これは、私たちのパートナーシップの第一歩です。

javac -cp src src/DevilBrowser.java src/OopsSearchEngine.java src/SettingsDialog.java

実行プロセス

ビルドが正常に完了したら、以下のコマンドでDevilBrowserを起動できます。新しい世界があなたを待っています。

java -cp src DevilBrowser
